
Factory Method ( patrón de diseño).

El factory method tiene como objetivo encapsular el código de creación 
de un objeto específico, centralizando la lógica en un solo punto.

 
 -- En sql para declararar un string se usan las '' comillas simples
 
 
-> java.sql.ResultSet:  a table of data representing a database result set, which is usually 
 generated by executing a statement that queries the database. 

A ResultSet object maintains a cursor pointing to its current row of data. Initially the cursor 
is positioned before the first row. The next() method moves the cursor to the next row, and 
because it returns false when there are no more rows in the ResultSet object,it can be used 
in a while loop to iterate through the result set.
